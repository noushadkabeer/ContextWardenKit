# Adding Providers

To add a new AI model provider to ContextWardenKit, please follow these steps:

1. Create a new file in `Sources/ContextWardenKit/Providers/`
2. Implement the `AIToolProvider` protocol.
3. Add corresponding unit tests in `Tests/ContextWardenKitTests/`.
4. Ensure no third-party dependencies are added.
5. Update `CHANGELOG.md`.
